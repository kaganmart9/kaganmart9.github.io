Demand Forecasting Datathon - Details

Technologies:

Python, Pandas, AutoGluon, StatsForecast (TSB), Feature Engineering, Time-Series Segmentation

Description:

Participated in a demand forecasting datathon organized for a leading European home appliances manufacturer. Developed a complete end-to-end solution that generates 12-month SKU and category-level forecasts using a hybrid approach: AutoGluon for regular products, StatsForecast TSB for intermittent demand, and simple mean for new launches. Implemented automatic product segmentation, phase-out detection, extensive feature engineering, custom RWMAPE metric, and hierarchical aggregation. Ranked among the top solutions with a production-ready pipeline.

Pharmaceutical Sales Forecasting Details

Technologies:

Python, Pandas, NumPy, Scikit-learn, Matplotlib/Seaborn, Time Series Analysis Libraries

Description:

End-to-End Telco Churn Analytics & BI Solution - Details

Technologies:

Microsoft SQL Server, T-SQL, DBeaver, Power BI Desktop, DAX, Star Schema Data Modeling

Description:

Developed a full end-to-end business intelligence solution for telecom customer churn analysis using the IBM Telco dataset. Built a robust ETL pipeline: ingested raw CSV data, performed cleaning and feature engineering (e.g., tenure cohort segmentation), and implemented a star schema data warehouse with dimension (customer, plan, payment) and fact tables. Created interactive Power BI dashboards with DAX-calculated KPIs revealing 26.54% overall churn rate, $139K churned MRR (30.5% of total), and high-risk segments (new customers, Fiber Optic users, Electronic Check payers). Provided actionable retention recommendations based on segment-specific insights.
